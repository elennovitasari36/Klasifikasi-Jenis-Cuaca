## 📌 Tentang Project

Project ini merupakan implementasi Data Mining dengan studi kasus **Klasifikasi Jenis Cuaca** menggunakan pendekatan Machine Learning. Dataset yang digunakan bersumber dari Kaggle (Weather Type Classification) dengan 13.200 data dan 4 target klasifikasi: **Rainy, Sunny, Cloudy, dan Snowy**.

Seluruh proses mengikuti framework **CRISP-DM**, meliputi:
- Business Understanding
- Data Understanding
- Data Preparation
- Modeling
- Evaluation
- Deployment

Algoritma yang digunakan adalah **Random Forest** yang dioptimalkan dengan **GridSearchCV**, menghasilkan akurasi sebesar **93.57%**, melampaui target minimum 80%.
